# 剑指offer刷题说明

刷题地址: http://www.nowcoder.com/ta/coding-interviews?page=1

需要找到最优解法：参考学习牛客网**左程云**视频、牛客网算法讨论、《剑指offer》图书，LeetCode等。

直接编辑器敲出代码，或在纸上直接写出代码（注意时间/空间复杂度）；剑指offer整体难度都不高，不过如果手撕代码的话，还是很锻炼的。
