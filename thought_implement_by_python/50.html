50 数组中重复的数字
Table of Contents
  • O(n)求解

O(n)求解

主要是讲正确的位置放上正确的数

  • 例1
   [2, 1, 3, 1, 4]
=> [3, 1, 2, 1, 4]
=> [1, 1, 2, 3, 4]
=> 1 重复
  • 例2
   [2, 3, 1, 0, 2, 5, 3]
=> [1, 3, 2, 0, 2, 5, 3]
=> [3, 1, 2, 0, 2, 5, 3]
=> [0, 1, 2, 3, 2, 5, 3]
=> 2 重复

# -*- coding:utf-8 -*-
class Solution:
    # 这里要特别注意~找到任意重复的一个值并赋值到duplication[0]
    # 函数返回True/False
    def duplicate(self, numbers, duplication):
        # write code here
        n = len(numbers)

        i = 0
        while i < n :
            if numbers[i] == i :
                i += 1
            else:
                if numbers[numbers[i]] == numbers[i] :
                    duplication[0] = numbers[i]
                    return True
                tmp = numbers[numbers[i]]
                numbers[numbers[i]] = numbers[i]
                numbers[i] = tmp

        return False
